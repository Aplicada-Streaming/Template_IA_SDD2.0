# 3. Instrucciones de Uso

1. Copiar este template y renombrarlo como `sprint-XX-retrospectiva_v1.0.md`.
2. El facilitador prepara la sesión y modera la discusión.
3. Registrar las aportaciones del equipo en las secciones correspondientes.
4. Priorizar las acciones de mejora (máximo 2-3 por sprint para asegurar seguimiento).
5. Asignar responsables y fechas de compromiso para cada acción.
6. Revisar las acciones del sprint anterior y actualizar su estado.
7. Compartir el documento con todo el equipo tras la sesión.
