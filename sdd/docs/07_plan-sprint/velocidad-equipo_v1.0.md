# 3. Velocidad Promedio

**Pendiente de cálculo.**

La velocidad promedio se calculará una vez completados al menos 3 sprints con datos de puntos completados.

---

# 4. Notas

- Completar la columna "Puntos completados" al finalizar cada Sprint Review.
- La velocidad promedio se calcula con los **últimos 3 sprints completados**.
- Utilizar la velocidad promedio como referencia para el Sprint Planning (no como compromiso fijo).
- Las variaciones significativas (>20%) deben analizarse en la retrospectiva.
